Comandos de diagnóstico

Ejecuta estos comandos en la Raspberry Pi para diagnóstico avanzado: 

# Ver conexiones WiFi guardadas
nmcli connection show

# Ver estado de interfaces
nmcli device show

# Ver logs de NetworkManager
journalctl -u NetworkManager -f -n 50

# Ver log de WiFi
cat /var/log/wifi_connect.log
